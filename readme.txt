[ENGLISH VERSION]
Welcome to AristoMaps,
an online card game where players compete using cards with physical attack, 
magic power, and physical defense stats. The goal is to reduce the opponent's 
prestige points and defend his own.

This game has been designed with a "retro game" type framework and offers a fun 
and challenging gaming experience. Please read this readme file carefully for detailed 
installation instructions and game rules.

========================
   Getting-started
========================
To start a game of AristoCartes, please follow the steps below:

1. Launch the program and choose your preferred language for the user interface.

2. Select your game mode: 1 player (AI) or 2 players.

3. Read and accept the rules by entering "ok".

4. Choose your game strategy.

5. Select your cards one by one to fill your deck. If you are playing in 1 player mode, 
the AI will select the cards for the opponent.

5. When you have finished selecting your cards, the battle begins. 
Cards fight each other card by card, follow the on-screen instructions to play.

Enjoy your game of AristoCartes!

NB: Don't forget to save your games so you can continue playing later.
PS: You can add as many cards as you want in the cards.txt file
     (Attention, do not touch the first line!)

[VERSION FRANÇAISE]
Bienvenue dans AristoCartes, 
un jeu de cartes en ligne où les joueurs s'affrontent en utilisant des cartes avec 
des statistiques d'attaque physique, de puissance magique et de défense physique. 
Le but est de réduire les points de prestige de l'adversaire et de défendre les siens. 

Ce jeu a été designé avec un cadre type "jeu retro" et offre une expérience de jeu 
amusante et stimulante. 
Veuillez lire ce fichier readme attentivement pour des instructions 
d'installation et des règles de jeu détaillées.

========================
  Démarrage  
========================
Pour démarrer une partie de AristoCartes, veuillez suivre les étapes suivantes :

1. Lancer le programme et choisissez votre langue préférée pour l'interface utilisateur.

2. Sélectionnez votre mode de jeu : 1 joueur (IA) ou 2 joueurs.

3. Lisez et acceptez les règles en entrant "ok".

4. Choisissez votre stratégie de jeu.

5. Sélectionnez vos cartes une par une pour remplir votre paquet. Si vous jouez en mode 1 joueur, 
l'IA sélectionnera les cartes pour l'adversaire.

5. Lorsque vous avez terminé de sélectionner vos cartes, le combat commence. Les cartes se battent 
entre elles carte par carte, suivez les instructions à l'écran pour jouer.

Profitez de votre partie de AristoCartes !

NB : N'oubliez pas de sauvegarder vos parties pour pouvoir continuer à jouer plus tard.
PS : Vous pouvez ajouter autant de cartes que vous voulez dans le fichier cartes.txt 
    (Attention, ne pas toucher à la première ligne !)
